# How to Use This Docker Setup

## Build

```text
(host)$ sh build.sh
```

## Rebuild

```text
(host)$ sh rebuild.sh
```

## Start the container

```text
(host)$ sh up_docker.sh
```

## Attach to the container

```text
(host)$ sh exec_zsh.sh
```

## Stop the container

```text
(host)$ sh down_docker.sh
```
